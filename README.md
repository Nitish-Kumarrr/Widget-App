# Widget-App
**Deployed Link:   https://nitishkumar-widgetapp.netlify.app/**
To Locally run the app create a react app using vite and then clone the repository in your system and install all the packages:-
1.Install Tailwind with Vite
2.Install react-redux , @reduxjs/toolkit,redux-persist
3.Install react-toastify
Run the application

Widget-App is a fully Responsive Web-App. In this app you can dynamically add and remove widget from a section.Implement Search functionality where we can search for particular widget.I have used Redux store management tool to add and remove data.
